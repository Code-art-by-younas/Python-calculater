# 🧮 Advanced Python Calculator

This is a simple but advanced Python calculator project built using basic Python concepts.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division (with zero error handling)
- Power operation
- Modulus operation
- Continuous loop (no need to restart program)

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Download the file
2. Open terminal or VS Code
3. Run command:
python calculator.py

## 📌 Purpose of Project
This project is made for learning Python basics and logic building for beginners and GitHub portfolio.
